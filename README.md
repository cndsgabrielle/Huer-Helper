# Huer Helper

Huer Helper is an Android application built to assist individuals with color blindness and visual impairments in identifying and matching colors in real-time.

### Key Features
* **Live Color Scanning**: Identify any color via camera with instant Hex/RGB data.
* **Style Matcher**: Check if two colors clash or complement each other.
* **Personal Bookmarks**: Save your favorite scans and matches to your library.


* **Language**: Kotlin
* **UI**: XML Layouts with Material Design
* **Database**: Local storage for bookmarked colors
